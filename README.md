📊 Telecom X — Parte II
Previsão de Evasão de Clientes (Churn)



📌 Sobre o Desafio

Este projeto faz parte do desafio Telecom X – Parte 2: Prevendo Churn, no qual o objetivo é desenvolver modelos de Machine Learning capazes de prever quais clientes possuem maior probabilidade de cancelar seus serviços.

Após a etapa de análise exploratória realizada anteriormente, o foco desta fase é construir um pipeline de modelagem preditiva, permitindo que a empresa identifique clientes com risco de evasão e atue de forma preventiva.



🎯 Objetivos do Projeto

Explorar modelos de Machine Learning e desenvolver modelos capazes de prever quais clientes têm maior chance de cancelar seus serviços na empresa fictícia TelecomX com os dados previamente tratados.

Python

Pandas

NumPy

Matplotlib

Scikit-learn

📂 Estrutura do Projeto

README.md: Arquivo com descrição do projeto.

TelecomX_parte2_BR.ipynb: Notebook contendo todas as células de execução do projeto e relatório final.

dados_tratados.csv: Base de dados tratada na parte 1 do projeto.



🔎 Etapas do Projeto



1️⃣ Pré-processamento dos dados

Tratamento de variáveis

Encoding de variáveis categóricas

Normalização de dados

Divisão entre dados de treino e teste



2️⃣ Análise de correlação

Foi realizada análise de correlação entre variáveis numéricas para identificar relações com a variável Churn e apoiar a seleção de atributos relevantes para os modelos.



3️⃣ Modelagem preditiva

Foram treinados diferentes algoritmos de classificação:

Dummy Classifier (baseline)

Árvore de Decisão

KNN

Regressão Logística

SVM

Random Forest



4️⃣ Avaliação dos modelos

Os modelos foram avaliados utilizando métricas de desempenho para comparar sua capacidade de prever clientes com maior risco de evasão.
